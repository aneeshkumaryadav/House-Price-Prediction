About the project

HousePriceAI reads a property's vitals — area, rooms, parking, furnishing, and locality — and returns a defensible number in seconds, the same way an appraiser would, minus the wait.

It's a regression model trained on {{ "{:,}".format(stats.rows) }} real Indore-area listings, wrapped in a small web app: enter a few details or upload your own spreadsheet, and get prices with a confidence range and the features that drove them.
